# Logistic Regression using Python

This project demonstrates a simple implementation of Logistic Regression using Python and Scikit-learn.

## Project Overview

This project was created to strengthen my understanding of classification algorithms in machine learning.

Logistic Regression is a supervised machine learning algorithm used for binary classification problems.

In this project:
- The dataset is loaded
- Data is split into training and testing sets
- A Logistic Regression model is trained
- Model accuracy is evaluated
- Classification results are visualized

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Output
The model predicts class labels and visualizes the decision boundary.
